# 🛍️ Customer Shopping Behavior Analysis

An end-to-end data analytics project that analyzes customer shopping behavior using Python, PostgreSQL, SQL, and Power BI. The project focuses on transforming raw retail transaction data into meaningful business insights through data cleaning, exploratory analysis, SQL querying, and interactive visualizations.

---

## 📌 Project Overview

The objective of this project is to understand customer purchasing patterns, product performance, and subscription behavior using a retail dataset containing **3,900 customer transactions**. The insights generated can help businesses improve marketing strategies, customer retention, and sales performance.

---

## 🛠️ Tech Stack

* Python (Pandas, NumPy)
* PostgreSQL
* SQL
* Power BI
* Jupyter Notebook

---

## 📂 Dataset

The dataset includes customer demographics, purchase details, product categories, subscription status, shipping methods, discounts, review ratings, and purchase history.

* **Rows:** 3,900
* **Columns:** 18

---

## 📈 Project Workflow

### 1. Data Cleaning & Preparation (Python)

* Loaded and explored the dataset
* Handled missing values
* Standardized column names
* Created new features for analysis
* Checked data consistency
* Loaded cleaned data into PostgreSQL

### 2. Business Analysis (SQL)

Performed SQL analysis to answer business questions, including:

* Revenue by gender
* Customer spending patterns
* Top-rated products
* Shipping type comparison
* Subscriber vs. non-subscriber analysis
* Customer segmentation
* Product performance
* Revenue by age group

### 3. Dashboard Development (Power BI)

Created an interactive dashboard to visualize:

* Customer demographics
* Sales performance
* Product trends
* Subscription insights
* Customer segments
* Revenue analysis

---

## 💡 Key Insights

* Identified high-value customer segments.
* Compared spending behavior across customer groups.
* Evaluated the impact of subscriptions and discounts.
* Discovered top-performing products and categories.
* Analyzed customer loyalty and repeat purchases.
